PageRefresher

Описание принципов работы с расширением.

  • created: 03/22/2017
  • latest update: 03/22/2017
  • by: Vladislav Sereda

Данное расширения дает возможность совершать циклически повторяющиеся действия, что связаны с работой браузера. На данный момент есть поддержка 2 действий: 1) Перезагрузки / изменения URL страницы. 2) Бампа(отправки трейдов) на сайте csgolounge.com

Refresher

1) Секция Initial Time.

Данная секция задает начальное время для запуска работи расширения.

За основу берется сегодняшняя дата.

В полях устанавливаются:

Hours - часы.

Minutes -  минуты.

Seconds - секунды.

2) Секция Refresh Time.

Данная секция задает частоту для перезагрузки страницы.  

В полях устанавливаются:

Hours - часы.

Minutes -  минуты.

Seconds - секунды.


3) Секция TargetURL.

Задает ссылку, для открытыя страницы, если перезагрузка происходит на :
4, 9, 14, 19, 24, 29, 34, 39, 44, 49, 54, 59 минутах.

Bumper

Работа бампера не отображаеться во всплывающем окне.

Работу бампера необходимо настраивать через окно настроек.

   По умолчанию Бампер срабатывает каждый раз, начиная с Initial Time, указаного во всплывающем окне. Бампер повторяет свои действия каждие Refresh time секунд, что указываються во всплывающем окне.

   Можно задать сдвиг для времени запуска бампера через окно настроек. Изменение Refresh time для бампера не предусмотрено.

Settings

Окно настроек, позволяет настроить детали работы расширения.

Состоит из следующих секций)

1) Общие настройки

Флажок "Разрешить автозапуск расширения" позволяет расширению начинать свою работу автоматически, используя синхронизированые с профилем гугл настройки, на страницах, url которых указан у поле "Набор страниц, где расширение будет запускаться автоматически".

Поле "Набор страниц, где расширение будет запускаться автоматически" . 
Правила заполнения:
1. Указываеться частично или полностью ссылка на страницу, где расширение планируеться запускать автоматически, например http://google.com/ или csgolounge.

2. Если необходимо указать несколько страниц, они указываються через запятую.

3. Во избежание ошибок не рекомендуется давать имена вида (№. #тутБылЯ, с).

4. Все что не является пробельным символом либо запятой расцениваеться как URL адрес и будет использоваться для работы расширения.

2) Свойства рефрешера

Блок "Установить задержку до перезагрузки".

В данном блоке указываеться время, на которое необходимо сдвинуть перезагрузку страницы назад во времени, относительно времени, что указывается во всплывающем окне.

То есть если время для начала работы во вспылвающем окне указано как 19:00:00, а в полях данного блока указано 0:1:0 , то время для начала рефреша сдивнется на 1 минуту назад и станет равно 18:59:00.

Не рекомендуется указывать в полях отрицательные значения.

3) Свойства бампера

Блок "Режим бампа сделок".

Флажок устанавливет интервал между бампами в 10 секунд. Если он не установен, интервал - 0 секунд.

Блок "Установить задержку до бампа".

В данном блоке указываеться время, на которое необходимо сдвинуть бамп трейдов на csgolounge  назад во времени, относительно времени, что указывается во всплывающем окне.

Принцып работы идентичен блоку "Установить задержку до перезагрузки".
